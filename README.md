# Weight-Aircraft-Forecasting

### Introduction

The main purpose of this repository is to forecast the weight that one aircraft would have at the Take-Off. In addition, the methodology used is going to be as simpler as possible due to the fact that the main idea could be running this code on a streaming data in a Big Data Plataform. Thus, an easy data dealing could minimize cost to the company besides to meet the requirements of the job.

### Data source

The dataset used was collected from the measures of a flying company has done in order to study the problem above exposed. It implies that it is a real problem and for this reason I am not going to upload the data here.
